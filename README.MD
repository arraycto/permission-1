# 权限控制的后台管理轮子

# 简介
  用前后端分离的方式vue+springboot搭建的只带权限控制的开发基础模板
  代码生成、日志记录，参数校验、权限控制

# 涉及

#### 后端：

  springboot 2.2.4.RELEASE

  mybatis 

  mybatis-plus 3.3.1.tmp

  mysql 8.0

  spring cache

  spring security 5

  knife4j-spring-ui 2.0

  hutool 5.1.0

  p6spy 3.87  sql执行速度检测

  ....

#### 前端:

  vue 2

  vue-cli 4.22

  vuex

  axios

  element-ui 2.13.0

  .....
